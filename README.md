# Work Process

## Prerequisites

1. Github Repo for the Project
2. Qwen Coder connected to Github Repo
3. Local IDE, VS Code, connected to same Repo
4. Google Chrome

## Development Workflow

1. Discuss the changes with Qwen Coder
2. After the changes are good, Publish to Github as PR
3. Merge the PR from Github
4. Sync the codebase in your local machine, VS Code maybe
5. Reload the extension on Chrome
6. Reload the page you want the extension to work
